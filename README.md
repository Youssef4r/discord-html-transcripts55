# 📋 نظام نسخ التذاكر - عائلة العمدة

نظام متقدم لعرض نسخ التذاكر من ديسكورد بشكل منسق وجميل.

## ✨ المميزات
- عرض جميع رسائل التذكرة
- عرض الامبيدات والمرفقات
- تحويل mentions لأسماء المستخدمين
- دعم العربية بالكامل
- إحصائيات كاملة للتذكرة
- إمكانية الطباعة والتحميل

## 🔗 الروابط
- معاينة التذكرة: `?data=...` أو `?id=...` أو `?url=...`
- تحميل التذكرة: إضافة `&download=true`

## 📞 الدعم
للاستفسارات والدعم، تواصل مع فريق الدعم الفني.
